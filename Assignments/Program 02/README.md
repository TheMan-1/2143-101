## A02 - Program 02 (SHAPES BUT PRETTY)
### Gage Woodall
### Description:

Loads Game of Life shapes from a JSON file and draws the chosen shape in an SDL2 window instead of the console

### Files

|  #  | File            | Description                                       |
|:---:|-----------------|---------------------------------------------------|
|  1  | main.cpp        | Uses SDL2 + json.hpp to render a shape on screen  |
|  2  | shapes.json     | Shape data (names, sizes, and cell coords)        |
|  3  | Program_02.exe  | Compiled executable                               |
| ETC | Files From Before                                                   |

### Instructions

- Keep all files in the same folder
- Open the **MSYS2 MinGW64** shell.
- `cd` into the `Program_02` folder.
- Build with:
 make run
