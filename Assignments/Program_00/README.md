## A00 - Program 00 (SDL2 Window)
### Gage Woodall
### Description:

Opens a window titled **“Game of Life — Program 00”** and fills it with a maroon background.

### Files

|  #  | File            | Description                                  |
|:---:|-----------------|----------------------------------------------|
|  1  | main.cpp        | Creates the SDL2 window and renderer         |
|  2  | Makefile        | Build script using                           |
|  3  | DejaVuSans.ttf  | Font file kept with the program              |
|  4  | program00.exe   | Compiled executable                          |

### Instructions

- Build in the **MSYS2 MinGW 64-bit** shell.
- Very finicky and annoying.
- Runs for a few seconds with a maroon background.
