## A01 - Program 01 (SHAPESSSSS)
### Gage Woodall
### Description:

Loads Game of Life shapes from a JSON file and prints the chosen shape to the console.

### Files

|  #  | File          | Description                                  |
|:---:|---------------|----------------------------------------------|
|  1  | main.cpp      | Reads shapes.json and prints a shape grid    |
|  2  | shapes.json   | Shape data (names, sizes, and cell coords)   |
|  3  | json.hpp      | Single-header JSON library used by main.cpp  |
|  4  | program01.exe | Compiled executable                          |
| ETC | Files From Before                                            |

### Instructions

- Keep all files in the same folder.
- Build with any C++17 compiler (include **json.hpp and shapes.json**).
- Run the program, type a shape name from the list, and it prints the shape.
